HTML Tags Documentation

A comprehensive guide to HTML tags with examples.

Basic Tags

<h1> to <h6>

Defines HTML headings.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

Formatting Tags

<strong>

Defines bold text.

<strong>This text is bold.</strong>

<em>

Defines italic text.

<em>This text is italic.</em>

Interactive Tags

<button>

Defines a clickable button.

<button onclick="alert('Clicked!')">Click Me</button>
